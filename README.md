    * unconnected
      * command : 장기미사용 등록 
      * query : 장기미사용 조회
  * web :http 처리 요청 
    * v1
      * admin 
* 특정사용자(프로젝트에서는 3개월 미로그인 사용자)를 조회하여 맴버테이블에서는 소프트딜리트를 사용하여 업데이트하고 미사용테이블에 입력하는 방법을 구성하였습니다.
* 대용량처리를 위해 병렬 처리를 사용하였습니다. (ForkJoinPool)
